Innehållspaneler

Används för informationsgruppering eller statusblock. Diamanten överst fylls med panelens färgkategori när användaren hovrar över ytan.

Accent Panel

Framhäver viktig information med Wigells primärfärg. Används sparsamt.

Neutral Panel

Innehåller standarddata, instruktioner eller övergripande funktionell information.

Positiv Panel

Indikerar lyckade processer, systemfördelar eller slutförda steg i ett arbetsflöde.

Negativ Panel

Drar uppmärksamhet till varningar, kritisk data eller destruktiva val.

Diskret Panel

Används för mindre viktiga meddelanden eller information som inte ska framhävas.

CSS & Designsystem

Kod- och Namngivningsstandard

För att hålla Wigells kodbas ren, skalbar och lättarbetad för alla utvecklare i teamet, följer vi dessa grundläggande principer:

Namngivning (CSS)
  • Komponentbaserat: Vi utgår från ett grundblock (t.ex. .btn, .panel).
  • Modifierare: Varianter av en komponent prefixas med grundnamnet (t.ex. .btn-primary, .panel-negative).
  • Små bokstäver och bindestreck: Klasser skrivs alltid med gemener (kebab-case), t.ex. .nav-links.
Kodstandard
  • Använd variabler: Hårdkoda aldrig färger eller animationstider. Använd alltid våra CSS Custom Properties (ex. var(--accent)).
  • Semantisk HTML: Använd rätt HTML5-taggar (<nav>, <section>) istället för oändliga <div>-element.
  • Responsivt först: Använd relativa enheter (rem, %) och media-queries för att säkerställa att komponenten fungerar på alla skärmar.

Rot-variabler

Klistra in detta block i toppen av din huvudstylesheet. Alla Wigell-komponenter förutsätter att dessa variabler finns tillgängliga globalt via :root.

--bg Sidans huvudbakgrund

--bg-code Kodblock och terminalbakgrund
--accent Wigell Koppar — primär varumärkesfärg
--accent-hover Koppar hover-tillstånd
--color-function Cyan — funktionella åtgärder
--color-positive Smaragd — framgång, bekräftelse

Diamant-geometrin

Wigells visuella signum är den 45°-roterade kvadraten. Den återkommer i logotyp, knappar, navigering, paneler och formulärkontroller. Implementeras alltid med transform: rotate(45deg).

Fylld — aktiv/vald

Kontur — tillgänglig

Dämpad — inaktiv


        /* Grundläggande diamant-form */
        .diamond {
            width: 16px; height: 16px;
            transform: rotate(45deg);
            background: var(--accent);
        }
        
        /* Diamant med hover-rotation */
        .diamond::before {
            content: '';
            width: 10px; height: 10px;
            transform: rotate(45deg);
            transition: all var(--transition);
        }
        .diamond:hover::before { transform: rotate(225deg) scale(1.3); }

Övergångar

Använd alltid --transition-variabeln för konsekventa animationer. Kurvan ger ett snabbt intryck och ett mjukt avslut.


        /* Använd variabeln — hårdkoda aldrig övergångsvärden */
        .element {
            transition: all var(--transition);
        }
        
        /* Samma som: */
        .element {
            transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
        }
